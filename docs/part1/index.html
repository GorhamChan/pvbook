使用功能介绍

为防止本软件遭到滥用，对于一些特别接口进行了隐藏，如需使用需要通过“更多-->嗅探”输入口令“pvtoken=gorhamchan2021”，点确定后重启APP可进行解锁。

如果你的网络是湖北移动网络，可通过“更多-->嗅探”输入口令“hbyd”，点确定后重启APP解锁IPTV入口。
