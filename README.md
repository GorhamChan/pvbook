# 简介

ParseVideo是一个学习项目，主要学习安卓开发和反编译技术。软件借鉴了GetVideo部分内容，目前软件几乎覆盖支持市面上大部分主流音视频网站的搜索或解析功能。此款软件仅为学习研究用途，请勿将此软件用于不当用途，由此产生的后果与本人无关，如有其它问题请联系[GorhamChan@163.com](mailto:GorhamChan@163.com)。
